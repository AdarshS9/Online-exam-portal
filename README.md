# ExamPro - Online Quiz & Exam Platform

A professional-grade, full-stack online examination platform featuring role-based access, real-time assessments, and premium glassmorphism UI.

## 🚀 Quick Access (Development)
The application is currently available on the local network at:
**[http://192.168.1.17:5175](http://192.168.1.17:5175)**

## 🛠️ Technology Stack
- **Frontend**: React, Vite, Lucide React, Chart.js
- **Backend**: Node.js, Express
- **Database**: Turso (libSQL)
- **Styling**: Vanilla CSS (Premium Glassmorphism)

## 📖 How to Run
1. Clone the repository.
2. Run `npm install` in both `client` and `server` directories.
3. Use the `run_project.bat` file or run `npm run dev` in the root directory.
